Big Data

O que é Big Data?




Big Data descreve um imenso volume de dados, estruturados ou não, que impactam os negócios no dia a dia.

A quantidade não importa, mas sim o que as empresas fazem com os dados importantes dentre esses.

Big Data pode ser analisado para a obtenção de insights que levam a melhores decisões e direções estratégicas de negócio.

História

História

O termo é novo, porém o conceito de recolher e armazenar enormes quantidades de dados para avaliações posteriores é antigo.

História



Esse nome “Big Data”, ganhou força em 2000, quando o analista Doug Laney, articulou 3 Vs: volume, velocidade e variedade.

• Volume: coleta de informações de uma variedade gigante de fontes, isso inclui redes sociais, transações, ou transmissões de maquina a máquina feitas por sensores.

• Velocidade: os dados fluem numa velocidade muito alta e sem dificuldades.

• Variedade: os dados são gerados em diferentes formatos, numéricos, estruturados, e-mails, vídeos, transações financeiras.

Como funciona?

Antes de descobrir como big data pode favorecer o seu negócio, você deve primeiro entender de onde isso vem. As fontes de big data geralmente vem de uma dessas três categorias:

  • Transmissão de dados (streaming data)

    Inclui dados que chegam aos seus sistemas de TI a partir de uma rede de dispositivos conectados.

  • Dados de redes sociais

    Os dados são um conjunto cada vez mais atraente de informação, particularmente para marketing, vendas e funções de apoio.

  • Fontes publicamente disponíveis

    Quantidades de dados estão disponíveis por meio de canais públicos como o dados.gov.br do governo federal brasileiro

Após identificar as fontes de dados, é preciso considerar as decisões que precisará tomar, pois começará a fazer uso destas informações.

  • Como armazenar e gerenciar:

    Há opções de baixo custo para armazenamento de dados caso esta seja a melhor estratégia para o negócio

  • Quanto se deve analisar

    Determinar antecipadamente quais dados são relevantes antes de analisá-los ou não excluír dados

  • Como utilizar os insights descobertos

    É importante ter uma estratégia desenhada para quando tiver uma abundância de informações em mãos

A etapa final é procurar tecnologias que o ajudará a tirar melhor proveito da enorme quantidade de informações e das análises de big data

• Armazenamento barato de grandes quantidades de dados
• Processadores mais rápidos;
• Open source acessível e plataformas distribuidas de big data, como o Hadoop;
• Processamento paralelo, clusterização, MPP, virtualização, grandes ambientes de grid, alta conectividade e altas taxas de transferência;
• Computação em nuvem (cloud computing) e outros arranjos de alocação flexível de recursos.

Por que Big Data é importante?

A importância do big data não é quantidade de dados, mas o que você faz com eles. Quando se combina big data com a alta potência do analytics, você pode realizar tarefas relacionadas a negócios:

  • Causa de falhas

    determinar problemas e defeitos em tempo quase real

  • Gerar cupons de venda

    com base em hábitos de compra dos clientes

  • Recalcular carteiras de risco

    em questão de minutos

  • Detectar comportamentos fraudulentos

    antes que eles afetem sua organização

Soluções Big Data

Soluções Big Data



Lida com volumes muito grandes de dados variados, as soluções precisam trabalhar com distribuição de processamento e elasticidade, é o que suporta aplicações com grande volume de dados e que além disso, crescem em pouco tempo.

Os bancos de dados que exploram o modelo relacional, como o MySQL e o Oracle, não se mostram adequados a estes requisitos, pois são menos flexíveis, quanto maior um banco de dados se torna, mais custoso e trabalhoso ele fica: é preciso otimizar, acrescentar novos servidores, empregar mais especialistas em sua manutenção, enfim.

Soluções Big Data



O bancos de dados relacionais se baseiam em quatro propriedades que tornam a sua adoção segura e eficiente: Atomicidade, Consistência, Isolamento e Durabilidade.

• Atomicidade: toda transação deve ser atômica, isto é, só pode ser considerada efetivada se executada completamente
• Consistência: todas as regras aplicadas ao banco de dados devem ser seguidas
• Isolamento: nenhuma transação pode interferir em outra que esteja em andamento ao mesmo tempo
• Durabilidade: uma vez que a transação esteja concluída, os dados consequentes não podem ser perdidos.

GSMA Anuncia Novos Desenvolvimentos na Iniciativa Big Data para o Bem Social



Ensaios iniciais do Big Data para o Bem Social para epidemias e poluição ambiental estão atualmente em curso na Índia.

Nos ensaios, as operadoras estão captando indicadores em um formato consistente de resultados a fim de proporcionar percepções em padrões de movimento humano.

Estas percepções são então comparadas com fontes de dados de terceiros, como prevalência de doenças, infraestrutura e locais de recursos, poluição e dados meteorológicos, para apoiar agências, uma vez que elas tomam decisões sobre quando, onde e como implantar recursos.

A GSMA espera publicar os resultados destes ensaios no Congresso Mundial de Telefonia Móvel em fevereiro de 2018.
